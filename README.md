# RePlusbot

An all-around bot for discord servers

Forked from [Revoltbot](https://github.com/CsiPA0723/Revoltbot)

## Functionalities

- Moderation
- Spam, Profanity and link filter
- Server Currency 
- Music 
- Leveling sytem
- Auto reaction in media channels
- Registry system
- Warnings
- Keeps track of invites
